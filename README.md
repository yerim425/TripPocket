# MyTourList
- Android, 1인 제작

## 개요
### 내 주변 생활 인프라 장소 검색하는 앱입니다.
- 급하게 찾고 있는 장소를 빠르게 검색하여 볼 수 있습니다.
- 찜 기능을 통해 월할 때 바로 확인할 수 있습니다.

## 기술 스택
- 개발언어 : Java
- 네트워크 : Retrofit2
- DB : RoomDB
- 개발환경 : Android Studio, Visual Studio Code
- 오픈소스 : Kakao 로컬 검색 API

## 사용기술
- RoomDB
- Kakao 로컬 키워드 검색 API
- 무한 스크롤 페이징 구현

## 주요기능
### 1) 로그인&회원관리
### 2) 장소검색&필터링
### 3) 찜등록 및 조회
