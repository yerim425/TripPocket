# MyTourList
- Android, 1인 제작

## 개요
### 내 주변 생활 인프라 장소 검색하는 앱입니다.
- 급하게 찾고 있는 장소를 빠르게 검색하여 볼 수 있습니다.
- 찜 기능을 통해 월할 때 바로 확인할 수 있습니다.

## 기술 스택
- 개발언어 : Java
- 네트워크 : Retrofit2
- DB : RoomDB
- 개발환경 : Android Studio, Visual Studio Code
- 오픈소스 : Tour API

## 사용기술
- REST API
- 한국관광공사 API 활용
- HttpURLConnection / URL.openStream()
- XML Parsing
- 무한 스크롤 페이징 구현

## 주요기능
### 1) 지역 검색
### 2) 지역 상세 조회
